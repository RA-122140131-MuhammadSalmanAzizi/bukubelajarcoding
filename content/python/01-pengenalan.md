# Pengenalan Python

Python adalah bahasa pemrograman yang terkenal karena sintaksnya bersih dan mudah dibaca — mirip bahasa Inggris sehari-hari. Karena itu Python sering jadi bahasa pertama yang dipelajari, sekaligus jadi tulang punggung dunia **AI, data science, dan web backend**.

## Kenapa Python?

- **Mudah dibaca** — sedikit simbol, banyak kata yang masuk akal.
- **Serbaguna** — dari script kecil, web, sampai kecerdasan buatan.
- **Komunitas besar** — ribuan library siap pakai, dokumentasi melimpah.
- **Diminati industri** — salah satu bahasa dengan permintaan kerja tertinggi.

## Program Pertama

Program klasik "Hello, World" di Python cukup satu baris:

```python
print("Halo, Dunia!")
```

`print()` adalah perintah untuk menampilkan sesuatu ke layar. Apa pun yang ditaruh di dalam tanda kurung dan tanda kutip akan ditampilkan apa adanya.

## Menjalankan Python

Ada dua cara umum:

1. **File script** — tulis kode di file berakhiran `.py`, lalu jalankan lewat terminal:

```bash
python nama_file.py
```

2. **Mode interaktif (REPL)** — ketik `python` di terminal, lalu ketik kode baris per baris untuk uji coba cepat.

## Komentar

Komentar adalah catatan untuk manusia yang diabaikan oleh Python. Diawali tanda pagar:

```python
# Ini komentar, tidak dijalankan
print("Ini dijalankan")  # komentar bisa juga di akhir baris
```

> Catatan: Python sangat memperhatikan **indentasi** (spasi menjorok di awal baris). Ini bukan sekadar rapi-rapi — indentasi menentukan struktur program. Kita bahas lebih lanjut di pelajaran percabangan.

## Ringkasan

- Python dikenal karena mudah dibaca dan serbaguna.
- `print()` menampilkan teks ke layar.
- File Python berakhiran `.py` dan dijalankan dengan perintah `python`.
- Komentar diawali `#`.

Lanjut ke pelajaran berikutnya untuk mengenal **variabel dan tipe data**.
